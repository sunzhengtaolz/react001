#备注文档

1.搭建脚手架
  npm create-react-app react001
2.暴露webpack,安装less/less-loader依赖
  1）npm run eject #暴露webpack
  2）npm install less less-loader --save-dev #安装依赖
  3）修改webpack.config.dev.js webpack.config.dev.js文件
3.安装react-router-dom
  npm install react-router-dom --save-dev
4.安装ant design
  npm install antd --save-dev
5.安装redux
  npm install redux react-redux react-thunk --save
6.使用fetch，并构造和使用了nodejs服务端，设置请求响应成功
7.安装axios，并使用
  npm install axios
8.安装mysql支持模块
  npm install mysql --save
